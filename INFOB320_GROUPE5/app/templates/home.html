Bienvenue sur le site web du club

Description

Ce site est construit à partir de plusieurs technologies:

  • Flask
  • jQuery
  • Boostrap
  • Ajax
  • Jinja2

Il a pour but de représenter le site web d'un club de tennis de table. Il contient dès lors plusieurs pages représentant les différentes offres du club.

Le site a été créé dans le cadre du cours de Technologies web (INFOB320) de l'université de Namur
